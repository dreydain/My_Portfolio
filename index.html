Guess My Number

Code Website

I built this project as part of a Masters JavaScript Course I'm doing on Udemy. The instructor created all of the HTML and CSS code, I followed along with the JavaScript content. In this section of the course I practiced and solidified my understanding of JavaScripts DOM Manipulation, and learned the different variations of functions and how they operate.


Function Declarations: Called and captured the same way as Function Expressions, but function declarations CAN be called before they are defined in the code.

Function Expressions: Called and captured the same way as function declarations, but function expressions CANNOT be called before they are defined.

Arrow functions: Are similar to Function Expressions but are easier to write, they CANNOT be called before they are defined and DO NOT get their own 'this' keyword.

My job on this project:

Front-End Development and Deployment. This was my second project that I've deployed through Netlify.

Technical Skills Used:

JavaScript, Git, and Netlify.

Sport Coach

Code

A full-stack Sports Web App to manage custom fitness plans and workouts based on a specific sport/interest. There are two types of users, Coaches/Admin and Clients. Coaches can create custom fitness plans and workouts for their clients based on their specific sports/interests.
